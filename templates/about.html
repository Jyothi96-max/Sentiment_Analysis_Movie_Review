🎬 About the Movie Review Sentiment Analysis Project

This web-based sentiment analysis system allows users to input a movie review and classify it as Positive or Negative using Machine Learning and NLP techniques.

🎯 Project Objective

To analyze user-generated movie reviews using three different Naive Bayes models and compare their performance, providing real-time predictions through a user-friendly interface.

📁 Dataset

We use the IMDB Movie Reviews dataset, which includes thousands of labeled positive and negative reviews. It is one of the most commonly used datasets in NLP tasks.

🧹 Preprocessing Steps

  • 🔠 Tokenization: Breaking text into individual words
  • 🧽 Stopword Removal: Removing common, meaningless words like "is", "the", etc.
  • 🌱 Stemming: Reducing words to their root forms (e.g., "running" → "run")
  • 📊 Vectorization: Converting text into numerical vectors using Bag of Words

🧠 Machine Learning Models

🟩 Bernoulli Naive Bayes – Ideal for binary data (word presence)
🟨 Multinomial Naive Bayes – Performs well with word frequency
🟥 Gaussian Naive Bayes – Assumes continuous distribution (less suited for text)

⚙️ How It Works

  1. The user enters a review on the website.
  2. Selects one of the available models (BNB, MNB, GNB).
  3. The input is cleaned, processed, and vectorized using the trained pipeline.
  4. The selected model predicts the sentiment.
  5. The result is displayed with a confidence score.

💻 Technologies Used

  • 🔧 Flask for backend web routing
  • 📚 NLTK and Scikit-learn for NLP and ML
  • 🎨 HTML + CSS (with Flexbox) for UI design

🔍 Why This Project?

This project showcases the complete machine learning pipeline—from data preprocessing and model training to deployment. It's useful for students and developers learning NLP and real-world sentiment analysis.
